   ```bash

   ```

   ```bash
   git push origin feature/AmazingFeature
   ```

5. **Open a Pull Request**: Submit your changes for review.

## üé® Style Guide

To maintain the "Forge & Decay" aesthetic, please adhere to the following:

- **CSS Variables**: Always use the defined palette found in `index.html`:
  - `--forge-color`: `#e74c3c` (Red/Heat)
  - `--decay-color`: `#3498db` (Blue/Cool)
  - `--accent`: `#2ecc71` (Green/Growth)
- **Mobile First**: Ensure all components render correctly on mobile devices for field use.
- **Icons**: Use FontAwesome 6.4 standard icons.
